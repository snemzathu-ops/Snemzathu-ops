AgentCare SaaS Landing Page & CRM

Includes:
- Next.js Landing Page
- Lead Capture API
- SendGrid Email Automation
- Lead CRM with Status Pipeline
- Full RBAC Security
- Unauthorized Access Handling

Setup:
1. npm install @sendgrid/mail next-auth prisma
2. Add SENDGRID_API_KEY to .env
3. Run prisma migrate
4. Start app

Admin CRM:
/admin/leads
